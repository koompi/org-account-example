# Org Account

This is an example project for creating a web app that has:

    - users
    - orgs
    - auth roles

To run this project first you need to generate a mock database.

```Bash
    npm i && npm run init-db
```

Then run the database:

```Bash
    npm run start-db
```

Finally run the development

```Bash
    npm run start-app
```
